# CSS Circular Loading Spinner Animation

Um componente de carregamento (Loader/Spinner) moderno, minimalista e de alta performance desenvolvido exclusivamente com HTML5 e CSS3. O projeto apresenta uma animação rotacional contínua e fluida de um arco de progresso, ideal para ser integrada em telas de transição (Preloaders), carregamentos de páginas ou requisições assíncronas em sistemas web.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica simples do contêiner do carregador.
* CSS3: Estilização completa, uso de formas circulares, definição de bordas e gerenciamento de animações.
* Animações em CSS: Utilização de keyframes (`@keyframes`) para controlar a rotação contínua de forma otimizada para o navegador.

## Funcionalidades e Técnicas Aplicadas

* Design Minimalista e Limpo: Composição baseada em um anel circular com fundo cinza claro sutil e um arco de progresso em azul destacado, conferindo uma estética profissional e discreta à interface.
* Movimento Rotacional Infinito: Aplicação de regras de animação contínua (`infinite`) com interpolação linear (`linear`) para garantir que o spinner gire sem interrupções ou engasgos visuais.
* Técnicas de Bordas em CSS: Construção do arco de preenchimento utilizando propriedades de customização de bordas (`border` e `border-top-color` ou `border-left-color`) sobre um elemento com cantos totalmente arredondados (`border-radius: 50%`).
* Centralização Absoluta: Uso de CSS Flexbox para alinhar horizontal e verticalmente o spinner no centro exato da tela.
